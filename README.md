# 🎁 Cestas & Eventos Judith

Página web moderna y responsive desarrollada para un negocio de cestas personalizadas y eventos.

## 🚀 Tecnologías utilizadas

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion

## ✨ Características

- Diseño responsive
- Interfaz moderna y visual
- Integración con WhatsApp
- Galería de imágenes
- Branding personalizado
- Optimización visual y experiencia de usuario
